NAME 

vlibMimeMail - php class enabling you to send Mime encoded emails with attachments, html mail ..etc.

SYNOPSIS 

   include "./vLIB/vlibMimeMail.php";
   $htmlbody = "<html>\n<body>\nHello test!...\n\n</body>\n</html>";
   $mail = new vlibMimeMail; // create the mail
   $mail->to('to_example@example.com', 'Recipient 1');
   $mail->from('sender@example.com', 'Sender');
   $mail->subject("the subject of the mail");
   $mail->body("Hello test!... in plain text"); // set the body in plain text
   $mail->htmlBody($htmlbody); // set the body in html format
   $mail->cc("cc_example@example.com", 'Recipient 2');
   $mail->bcc("bcc_example@example.com", 'Recipient 3');
   $mail->priority(1); // set the priority to Highest
   $mail->attach("example.doc"); // attach word doc

   $mail->send(); // send the mail
   print("Message sent:<pre>".$mail->get()."</pre>");

If everything goes well, a mail should be sent to the recipients and you should see a page with the mail details.


DESCRIPTION 

vlibMimeMail is a class which makes sending mime encoded email messages a simple task. It will make your code alot
easier to read, write and debug.

vlibMimeMail includes the following features:

  • Send multiple attachments
  • Specify one or many recipients in To, Cc or Bcc
  • Specify a priority level
  • Auto validation of all email addresses
  • Request a receipt from the recipient
  • Write an HTML encoded message

MOTIVATION 

Almost all websites around need to send emails to their visitors at some point, whether it be a confirmation of joining
your site or a newsletter or something else.

This class simplifies the tasks of adding mime headers for all of the different features into easily understandable
functions.

autoCheck()  

  autoCheck (bool check)


If check is true (as it is by default), all email addresses passed to vlibMimeMail will be checked for validity.
It is just a syntax check thus is very quick.


to()  

  to (string email [, string name])

This function sets the 'To' header for the message. The email parameter is the email of the recipient and the name
parameter is the name of the recipient, the name parameter is optional.

You can call this function as many times as you like.

   $mail->to('to_example@example.com', 'Mr Example To');

attach()  

  attach (string filename [, string disposition, string mimetype])

This function attaches a file to the message.

Filename is the 'string' with the include path to the file you want to attach.
Disposition can be 1 of 2 values:
    attachment (default) - the file is attached to the message and displayed in the client as a link.
    inline - the client should try to display the attachment directly inside the message.
Mimetype, if specified, will set the mimetype for this particular attachment. If not specified it will search for the
mimetype (using the extension of the file) in vlibCommon/mime_types.php.

You can call this function as many times as you like.

   $mail->attach('./example.doc'); // simple attachment
   $mail->attach('./example.doc', 'inline', 'application/msword'); // full attachment



organization()  

  organization (string org)

This function sets the 'Organization' header for the message.

 

receipt()  

  receipt (bool yesno)

If yesno is set to true then the 'Disposition-Notification-To' will be set to either 'Reply-To' or 'From', depending
on which has been set.

 

priority()  

  priority (int level)

This function sets the 'X-Priority' header of the message.

Level must be between 1 and 5 and must be passed as an integer.
Levels are:
    1 - Highest
    2 - High
    3 - Normal
    4 - Low
    5 - Lowest

  $mail->priority(1); // sets the priority to highest

 

validateEmail()  

  validateEmail (string address)

This function validates the email address 'address' and returns true on validation success or false on failure.

  $mail->validateEmail('to_example@example.com'); // returns true
  $mail->validateEmail('to example@example.com'); // returns false

 

send()  

  send ()

This function compiles the email and sends it. Returns true on success, false on failure.

  $success = $mail->send();
  if (!$success) die('Mail not sent!');

 

get()  

  get ()

This function compiles the email and returns a string with exactly all of the informaton (headers and attachments)
as it will be sent by send().

  print('<pre>'.$mail->get().'</pre>');
