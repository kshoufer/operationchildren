vLIB Documentation        

vLIB is a Library of classes helpful to those developers looking to create real world applications.
All of the classes in this Library are subject to the OpenSource Artistic License.

Below is a list of the classes which link to their specific documentation.

vLIB

  • vlibTemplate - a template system to help you split the Layout and Logic of your PHP scripts.
  • vlibMimeMail - a class for sending mime-encoded emails with attachments, html body... etc.
  • vlibDate - a class for manipulating and displaying dates between years 0001-9999 in multiple languages.
